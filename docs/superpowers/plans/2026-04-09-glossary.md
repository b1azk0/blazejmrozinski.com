> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an SEO-first glossary section at `/glossary/[term]` with per-term structured data, cover images, and automated related-content linking.

**Architecture:** New `glossary` Astro content collection with markdown files per term. Two new pages (index + dynamic slug). Domain-to-color mapping in a shared utility. Cover image generation extended to handle glossary terms. New sitemap file. Footer link.

**Tech Stack:** Astro content collections, Zod schema, satori + resvg (cover images), Tailwind CSS, JSON-LD structured data.

```typescript
const GLOSSARY_DIR = join(ROOT, 'src/content/glossary');

function getGlossaryFiles(): Array<{ slug: string; title: string; domain: string }> {
  if (!existsSync(GLOSSARY_DIR)) return [];
  const files = readdirSync(GLOSSARY_DIR, { recursive: true })
    .filter((f) => String(f).endsWith('.md'));

  const terms: Array<{ slug: string; title: string; domain: string }> = [];
  for (const file of files) {
    const filePath = join(GLOSSARY_DIR, String(file));
    const content = readFileSync(filePath, 'utf-8');
    const fm = parseFrontmatter(content);
    if (!fm.term || !fm.domain) continue;
    if (fm.status === 'draft') continue;
    const slug = basename(String(file), '.md');
    terms.push({ slug, title: fm.term, domain: fm.domain });
  }
  return terms;
}
```

```typescript
import type { APIRoute } from 'astro';
import { getCollection } from 'astro:content';

const site = 'https://www.blazejmrozinski.com';

export const GET: APIRoute = async () => {
  const terms = (await getCollection('glossary', ({ data }) => data.status === 'published'))
    .sort((a, b) => a.data.term.localeCompare(b.data.term));

  const xml = `<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>${site}/glossary/</loc>
    <changefreq>weekly</changefreq>
    <priority>0.6</priority>
  </url>
${terms.map((term) => `  <url>
    <loc>${site}/glossary/${term.id}/</loc>
    <lastmod>${term.data.date.toISOString().split('T')[0]}</lastmod>
    <changefreq>monthly</changefreq>
    <priority>0.6</priority>
  </url>`).join('\n')}
</urlset>`;

  return new Response(xml, {
    headers: { 'Content-Type': 'application/xml' },
  });
};
```
