An autonomous content pipeline that transforms Semrush keyword data into a prioritized publishing backlog, generates SEO-optimized articles using AI, and publishes them to WordPress sites on a configurable cadence.

Features include cannibalization detection via embeddings, self-reviewing article quality scoring, and a WordPress plugin for site-level control.
